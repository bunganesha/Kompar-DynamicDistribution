## Deskripsi Program

Program ini merupakan simulasi dynamic distribution menggunakan Python dan multiprocessing.

Dynamic distribution adalah metode pembagian tugas secara dinamis kepada beberapa worker. Worker yang selesai lebih cepat akan langsung mengambil task berikutnya sehingga proses menjadi lebih optimal dan efisien.

Program ini dibuat untuk memenuhi tugas mata kuliah Komputasi Paralel dan Sistem Terdistribusi.
